🏥 Medico Medicine Store

System Design Documentation

1. System Overview

Medico is a comprehensive online medicine store that allows customers to browse, purchase, and order medicines with prescription support. The system includes user authentication, product management, shopping cart functionality, and prescription upload capabilities.

Key Features

  • User registration and authentication system
  • Product catalog with multiple categories (Tablets, Syrups, Supplements, Baby Products, Veterinary, Injections)
  • Shopping cart functionality with AJAX support
  • Prescription upload and management
  • Order processing with multiple payment methods
  • Admin panel for product and order management
  • Responsive design for mobile and desktop
  • Stock management system
  • Invoice generation and order tracking

3. Database Schema Details

USERS Table

CREATE TABLE users ( id INT AUTO_INCREMENT PRIMARY KEY, name VARCHAR(50), email VARCHAR(100), password VARCHAR(100) );

Purpose: Stores user account information for authentication and order management.

PRODUCTS Table

CREATE TABLE products ( id INT AUTO_INCREMENT PRIMARY KEY, name VARCHAR(100), price DECIMAL(10,2), category VARCHAR(50), stock INT, image VARCHAR(100) );

Purpose: Stores product information including pricing, stock levels, and images.

PRESCRIPTIONS Table

CREATE TABLE prescriptions ( id INT AUTO_INCREMENT PRIMARY KEY, user_email VARCHAR(100), file_path VARCHAR(255), uploaded_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP );

Purpose: Stores prescription file information for order validation.

ORDERS Table

CREATE TABLE orders ( id INT AUTO_INCREMENT PRIMARY KEY, customer_name VARCHAR(100), customer_email VARCHAR(100), product_id INT, quantity INT, total_price DECIMAL(10,2), payment_method VARCHAR(50), address VARCHAR(255), phone VARCHAR(20), order_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP, delivery_date DATE, prescription_path VARCHAR(255) );

Purpose: Stores complete order information including customer details, products, and prescription.

5. User Workflow

Customer Order Process

1
Browse Products
Customer navigates through different product categories (Tablets, Syrups, Baby Products, etc.)
2
Add to Cart
Customer adds desired products to cart using AJAX functionality
3
Review Cart
Customer reviews cart contents and proceeds to checkout
4
Authentication
If not logged in, customer is redirected to signup/login with checkout redirect
5
Checkout Form
Customer fills delivery details, payment method, and uploads prescription (if required)
6
Order Processing
System validates stock, processes payment, and generates order
7
Order Confirmation
Customer receives order confirmation with invoice and delivery details

6. Security Features

Authentication & Authorization

  • User registration with email validation
  • Password hashing using MD5 algorithm
  • Session-based authentication
  • Automatic logout functionality
  • Admin panel access control

Data Protection

  • SQL injection prevention through prepared statements
  • File upload validation (type and size restrictions)
  • XSS protection through output escaping
  • CSRF protection through session validation
  • Secure file storage with unique naming

Prescription Security

  • Automatic cleanup of old prescription files
  • File type validation (JPG, PNG, PDF only)
  • File size limits (5MB maximum)
  • Unique file naming to prevent conflicts
  • Prescription files linked to specific orders only

10. Conclusion

The Medico Medicine Store is a comprehensive e-commerce solution designed specifically for pharmaceutical retail. The system provides a secure, user-friendly platform for customers to purchase medicines while ensuring compliance with prescription requirements.

The modular architecture allows for easy maintenance and future enhancements, while the responsive design ensures optimal user experience across all devices. The admin panel provides complete control over products, orders, and user management.

With its robust security features, efficient order processing, and prescription management capabilities, Medico serves as a complete solution for online medicine retail operations.
